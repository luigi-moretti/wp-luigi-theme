Um tema WordPress moderno para desenvolvedores frontend com design limpo e responsivo.

== Description ==

O Frontend Dev Portfolio é um tema WordPress especialmente criado para desenvolvedores frontend que desejam mostrar seu trabalho de forma profissional e moderna. 

**Características principais:**

* Design moderno e limpo baseado em Tailwind CSS
* Totalmente responsivo para todos os dispositivos
* Custom Post Types para Habilidades e Projetos
* Formulário de contato funcional com AJAX
* Integração com WordPress Customizer
* Otimizado para SEO
* Compatível com Gutenberg
* Suporte a imagens destacadas
* Navegação suave entre seções
* Animações sutis e profissionais

**Seções incluídas:**

* Hero/Apresentação personalizada
* Sobre Mim
* Habilidades (gerenciadas via Custom Post Type)
* Blog integrado
* Formulário de contato
* Menu de navegação responsivo

**Personalização:**

O tema oferece opções de personalização através do WordPress Customizer, permitindo:

* Alterar nome do desenvolvedor
* Personalizar descrição
* Configurar email de contato
* Gerenciar habilidades e projetos

== Installation ==

1. Faça o upload do tema para a pasta `/wp-content/themes/`
2. Ative o tema através do painel 'Aparência > Temas' no WordPress
3. Personalize através de 'Aparência > Personalizar'
4. Adicione suas habilidades em 'Habilidades' no painel admin
5. Crie posts para seu blog

== Frequently Asked Questions ==

= Este tema é responsivo? =
Sim, o tema é totalmente responsivo e funciona perfeitamente em dispositivos móveis, tablets e desktops.

= Posso personalizar as cores? =
As cores podem ser personalizadas editando o arquivo style.css ou através de CSS adicional no customizer.

= O formulário de contato funciona? =
Sim, o formulário de contato está totalmente funcional e enviará emails para o endereço configurado no customizer.

= Como adiciono minhas habilidades? =
Vá em 'Habilidades' no menu administrativo do WordPress e adicione suas skills como posts.
